# お客様質問対応フローチャート（プロフェッショナルな言い換え集）

顧客対応における認識のズレを防ぎ、主導権を握りながら確実な回答を届けるためのフローです。

### STEP 1：意図理解（認識の同期）
- **アクション**: 質問を要約して復唱する。
- **フレーズ**: 「つまり、〇〇の条件における挙動についてのご質問、という認識で相違ないでしょうか？」

### STEP 2：質問の肯定（ポジティブな受容）
- **アクション**: 質問の重要性を評価する。
- **フレーズ**: 「非常に鋭いご指摘です。そのケースは運用上、極めて重要なポイントになりますね。」

### STEP 3：即答を避ける際の「プロの言い換え」
情報の正確性を担保するため、以下の理由を使い分けて慎重に対応します。

| シチュエーション | 推奨フレーズ |
| :--- | :--- |
| **情報の正確性重視** | 「情報のアップデートが走っております。誤認を招かないよう、最新のエビデンスを再確認させてください。」 |
| **専門知見による担保** | 「私個人の主観より、専門チームの確実な見解を添えて回答すべき案件だと判断しました。」 |
| **影響範囲の精査** | 「影響範囲が広いため、安易に回答せず、整合性を精査した上でお戻しさせてください。」 |
| **組織の決定を強調** | 「重要度が高いため、私の一存ではなく、社内コンセンサスを得た上での『正式回答』とさせてください。」 |

### STEP 4：出口戦略
- **アクション**: 期限を明確にし、QA管理表などに起票する。
- **フレーズ**: 「今の内容はQA管理表に起票し、〇日までに正式回答を差し上げます。よろしいでしょうか？」
